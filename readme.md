# Live Chat Example

Live chat usado como exemplo no DEVFEST NORDESTE 2016. :gift: :fire: 

## Installation

1. Clone repository

```bash 
$ git clone git@github.com:felipesousa/live-chat-app.git newApp
```

## Usage

1. Install dependencies

```bash 
$ cd newApp && npm install
```

2. Execute gulp task

```bash 
$ gulp
```
